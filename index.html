UseRoux

A web application built with Three.js to create an interactive guide to speed cubing using the Roux method. Walks users through solving a speed cube by going over each of the four steps of Roux in depth, showing an explanation and example solve for each step. Three.js provides an interactive cube you can examine and control. Allows users to walk through each step move by move.

App Tracker

A web application developed using React allowing users to catalog internship and job applications. Users can add applications by entering company/position name and any notes, then pressing 'Add Application'. Also stores number of interviews, if an offer was received, if a rejection was given and date of application. Has support for light and dark themes, exporting and importing applications, tracking application list statistics, and editing application entries. Sleek and responsive design using CSS3 and support for multiple devices using media queries.

Product Log

A cross-platform mobile utility developed using the flutter SDK. Allows users to document their purchases by either entering a UPC, scanning a barcode, or simply adding the product dynamically. Product information is stored in a local SQLite database and can be looked up, searched by name or tag, and sorted. Stored information includes UPC, picture, name, manufacturer, description, price, quantity, etc.

Pixel Striker

A mobile game written in the Godot game engine. Based around my love for soccer, play through three different game modes, free kicks, penalties, and goalie mode. In free kicks and penalties, you attempt to shoot the ball and score against an AI goalie. In goalie mode, you attempt to save the ball through swiping on the touchscreen. Google Play Games Services is integrated, granting 6 unique achievements and 3 unique leaderboards to compete on!

Minesweeper AI

My version of a minesweeper game written in vanilla javascript with a built in AI solver. The AI uses a probability map to determine which tiles are mines and which tiles are not. The AI works very well until the puzzles become very difficult (>90 mines).

Telegram Message Counter

A powerfully simple program written in Python to analyze texting trends. The software utilizes BeautifulSoup to scrape webpages generated by the Telegram Desktop App to collect data. It then displays this data using tkinter and matplotlib as barplots or line plots. It was originally written as a fun side project to analyze texting trends with my girlfriend.
