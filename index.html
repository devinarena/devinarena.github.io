About Me

My name is Devin Arena and I am a sophomore studying Software Engineering at Florida Gulf Coast University. I am fluent in C++, Java, C# and Python and enjoy writing useful apps and fun games.

You can find some of my side projects below as well as some contact info.

Projects

Most of my projects can be found on my GitHub.

Here are some of the ones I'm most proud of:

  • Production (Java)

    A full scale production project with database integration, employee login system, product line and production log functionality. Uses Object Oriented programming practices.

  • Telegram Message Counter

    A desktop application that utilizes BeautifulSoup to scrape and analyze a set of Telegram messages exported from the Telegram desktop app. Displays the number of messages per month and year on a bar plot or line plot using tkinter and matplotlib.

  • Minesweeper AI (JavaScript)

    An AI that plays a custom programmed version of MineSweeper in the browser. The AI solves the board using a probability map and has a high success rate once even a small area is uncovered.

  • Tic-Tac-Toe AI (JavaScript)

    An AI that plays the user in Tic-Tac-Toe in the browser. The AI uses the minimax algorithm to determine which move is best and almost never loses.
