About Me

I am currently a software engineering student at Florida Gulf Coast University. I love programming, designing games, tinkering with computers, playing the piano, and writing music. You can find some of my projects and ways to contact me below. In regards to programming, my interests lie in backend design and development.

Telegram Message Counter

A powerfully simple program written in Python to analyze texting trends. The software utilizes BeautifulSoup to scrape webpages generated by the Telegram Desktop App to collect data. It then displays this data using tkinter and matplotlib as barplots or line plots. It was originally written as a fun side project to analyze texting trends with my girlfriend.

ASCII Shooter

A small, fun game written in C++ that runs completely in the command line interface. Fight 6 different kinds of enemies and a final boss in a shooter game that utilizes ASCII characters for sprites. Developed using C++ and the Windows API.
